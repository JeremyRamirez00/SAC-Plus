# SAC-Plus 

Sistema de Atención al Cliente (SAC-Plus) desarrollado como proyecto académico.

## Descripción

SAC-Plus es una aplicación web que permite gestionar solicitudes de servicio técnico, permitiendo registrar, consultar, actualizar y finalizar solicitudes de clientes.

## Tecnologías utilizadas

- Java (Servlets)
- Apache Tomcat
- MySQL
- HTML, CSS, JavaScript

## Funcionalidades

-  Login de usuario
-  Registro de solicitudes
-  Visualización de datos (desde BD)
-  Búsqueda de registros
-  Actualización de información
-  Finalización de solicitudes

##  Base de datos

Ejecutar el archivo `database.sql` para crear la base de datos y las tablas necesarias.

##  Ejecución

1. Configurar Apache Tomcat
2. Importar el proyecto en IntelliJ
3. Configurar conexión a MySQL
4. Ejecutar el servidor
5. Acceder desde:
   http://localhost:8080/sacplus
6. Login admin-admin
